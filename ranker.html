Ranker

Abstract

  • App that allows you to order a list of items by pairwise ranking using ELO
  • For example, rank a list by importance
  • Or, rank a list by which order you want things to appear in
  • Using above two types of ranking, you can write an essay by ordering sentences in order they should appear, and also shorten or lengthen it with importance ranking and chopping off the least important sentences.
  • By using semantic relatedness, by answering comparison question of which of two items is more related a third item, you get a graph. You can then generate an essay/sequence by running shortest path algorithm on any two nodes in the graph. You can also see if there are unconnected graphs in the graph or other graph algorithms on this graph.
